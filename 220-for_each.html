FOR...EACH

O método forEach() em JavaScript é uma maneira conveniente de iterar sobre os elementos de um array. Ele executa uma função fornecida uma vez para cada elemento do array, facilitando a realização de operações como processamento, modificação ou exibição de dados.
A função forEach() não retorna um novo array, mas simplesmente executa a função fornecida para cada elemento do array.
Lembre-se disso, ele é um método do objeto Array aqui, não um loop como em outras linguagens, mas que acaba funcionando do mesmo jeito.
